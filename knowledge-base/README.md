# Knowledge Base

The cited grounding for the GTM Lab. The Wizard reads from here; every fact it teaches traces back to a
source listed below. This KB **grows as I work through the curriculum** — modules deepen as I learn them.

## What's here
- **`glossary.md`** — the language: ~90 GTM terms, defined tightly, each with a source and a verification
  flag (⚠ = treat as descriptive/contested, not a law).
- **`curriculum.md`** — the 9-module learning path: each module = a concept to learn + the source to ground
  it in + a runnable play + an explain-back checkpoint.
- **`topics/`** — deeper per-topic syntheses (added as each module is worked; currently being populated).

## Honesty rules for this KB
- Cite every substantive claim. Mark **settled/primary-sourced** vs **descriptive/contested** vs **unverified**.
- Several widely-repeated outbound "uplift" stats (e.g. "2–4×/5× signal-based reply") are **vendor-claimed
  and unverifiable** — they are flagged, not repeated as fact.
- Benchmarks reflect their sample (selection bias) — cite the source and the caveat, never as a universal law.
